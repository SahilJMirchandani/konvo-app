# Konvo App

Mobile app for managing projects and tasks built with React Native (Expo).

## Tech Stack
- React Native (Expo SDK 54)
- Redux Toolkit + Thunk
- Axios
- React Navigation

## Features
- OTP-based authentication
- Create, view, delete projects
- Create, complete, delete tasks
- Optional due dates on tasks
- Light/Dark mode support
- Protected routes with JWT

## Setup
1. Clone the repo
2. Run `npm install`
3. Update `src/api/axios.js` with your backend IP
4. Run `npx expo start`
5. Scan QR code with Expo Go (SDK 54)
